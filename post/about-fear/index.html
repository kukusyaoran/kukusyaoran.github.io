关于恐惧

2019-09-29 3 min read # 随笔

我们之前提出了一个问题，但并没有回答它：

人的恐惧到底来源于自身还是外部环境？

1. 对攻击性的解释：

人具有一种无意识的死的本能。人具有一种自我毁灭的无意识愿望。但是，一个具有健康人格的人是不会伤害自己的，所以这种自我毁灭的冲动就会无意识地转向外面，以攻击别人的方式表达出来。另一种解释，当人要达到自己目标的行为受阻时，就会产生攻击行为。一个处于巨大的挫折体验中的人，就可能出现比较持久的攻击行为。在大多数情况下，人并没有意识到自己攻击行为的真正原因。

2. 自由意识与决定论：

人的行为是环境影响与人所积累的经验的结果。

如何解释：

恐惧是一种意识而不是一种行为 --> 意识决定行为还是无意识决定行为？ --> 弗洛伊德对无意识力量控制的假设 --> 认知心理：很多信息的加工是在意识之下的水平上发生的 --> 恐惧的产生受外界信息来源与加工方式的影响，恐惧在很多情况下是在意识水平之下发生的。

受什么影响：

信息的来源 <= 渠道

信息的处理方式 <= 思维

3. 有关防御机制中的替代：

防御机制是人在处于焦虑感时，自我处理非预期想法和欲望的方法。替代将攻击性导入一个无威胁的目标物。当无法将冲动付诸于实际目标的时候，人会选择一个无威胁的目标物作为替代，比如生老公气的时候转向孩子发怒，这样会带来其他问题，但是向威胁小的人发怒，可以避免不被接受的想法被意识表达出来。弗洛伊德认为，人们的许多不合理的害怕或恐惧，都不过是象征性替代。替代只是防御机制中的一种。

4. 结论：

恐惧的产生是因为人具有一种无意识的死的本能，这种本能以攻击性的方式表达出来，一旦这种冲动无法得到释放，人就会处于焦虑感之中，为了缓解这种焦虑感，人就会寻找一种象征性的替代。恐惧的产生受外界信息来源与加工方式的影响，恐惧在很多情况下是在意识水平之下发生的。

5. 题外话：

一切营销都是指向人的无意识状态（网摘）

为什么学心理学？

因为想要知道为什么。明白人的各种心理情绪产生的原因，可以让自己免受心理问题永无止境的折磨，达到精神上的真正自由。最明显的一点是，你不用再问自己这样的问题「我为什么会这样，我明明不是这样的」，如果你已经有了这些概念，你就有了寻找答案的能力，而不是自责。
